ferramentas

NetBeans IDE, Adobe Illustrator

tecnologias

HTML5, CSS3, JavaScript (jQuery)

Criação do site para a turma 2013.2 do Curso Técnico em Informática do IFRN, apenas para praticar minhas habilidades recém aprendidas no curso.

A ideia do site é apresentar todos os alunos da turma através de uma placa de formatura virtual com direito a uma playlist das nossas músicas favoritas. Tudo construído com animações e transições do CSS3.

O site também conta com uma breve descrição de cada componente da turma.

E uma galeria com as nossas melhores fotos.

Guia de Estilo

Foi criado um pequeno guia de estilo para definir o visual de cada componente.

Mobile First

O site foi pensado para ser responsivo e acessado do smartphone.
